# Usage
<!-- usage -->
```sh-session
$ npm install -g @thebeyondgroup/shopkeeper
$ shopkeeper COMMAND
running command...
$ shopkeeper (--version)
@thebeyondgroup/shopkeeper/1.0.0 darwin-x64 node-v18.17.0
$ shopkeeper --help [COMMAND]
USAGE
  $ shopkeeper COMMAND
...
```
